### Скрипт для Violentmonkey (или Tampermonkey), который интегрирует в интерфейс страницы две дополнительные кнопки
- **Копирование ссылки** на текущий тайтл в форматированном виде `[Название](URL)` — готово для вставки в Markdown-документы, заметки.  
- **Переход на MyAnimeList** одним кликом — быстрый доступ к странице аниме на MAL.
